# Pixel & PDF

**Pixel & PDF** is a web-based file utility application built with **Python** and **Flask**. It provides fast, simple, and reliable tools for working with images and PDF documents through a clean, responsive interface.

## ✨ Features

### 📄 PDF Merger

* Merge multiple PDF files into a single document.
* Preserve the original page order.
* Download the merged PDF instantly.

### 🖼️ Image Converter

* Convert images between popular formats.
* Supports common image types such as JPEG, PNG, WEBP, BMP, and GIF.

### 🗜️ Image Compressor

* Reduce image file size while maintaining good visual quality.
* Simple one-click compression.
* Supports multiple image formats.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/pixel-and-pdf.git
```

Navigate to the project folder:

```bash
cd pixel-and-pdf
```

Install the required packages:

```bash
pip install flask pillow pypdf
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
